# GreenCampus Frontend (React without Vite)

This frontend uses React with `react-scripts` and connects to the Django API.

## Development

```bash
npm install
npm start
```

Runs on `http://localhost:3000` by default.

## Build

```bash
npm run build
```

## Environment

Create `.env` from `.env.example` and set:

```env
REACT_APP_API_BASE_URL=http://127.0.0.1:8000/api
```
